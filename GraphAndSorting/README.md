# GraphAndSorting

## Описание

Проект реализует обобщенную структуру данных "Граф" и алгоритмы сортировки на языке C\# (.NET 9.0). Решение включает консольное приложение для демонстрации и проект модульных тестов.

## Функциональность

### 1\. Граф (Graph\<T\>)

Реализация неориентированного графа на основе матрицы смежности.

  * **Хранение данных:** Использует `List<T>` для вершин и двумерный массив `bool[,]` для хранения ребер.
  * **Основные операции:**
      * Добавление и удаление вершин с динамическим пересчетом матрицы смежности.
      * Добавление и удаление ребер между существующими вершинами.
      * Проверка наличия вершин и ребер.
      * Расчет степени вершины (количество инцидентных ребер).
  * **Системные функции:** Реализованы `IEnumerable<T>`, `IEquatable`, `GetHashCode` и строковое представление в виде списков смежности.

### 2\. Алгоритмы сортировки (Sorting)

Статический класс, реализующий два метода сортировки:

  * **Cocktail Sort (Шейкерная сортировка):** Двунаправленная пузырьковая сортировка. Работает с массивами (`T[]`). Проходит по массиву слева направо и справа налево в одном цикле `do-while`.
  * **Strand Sort:** Рекурсивный алгоритм, извлекающий отсортированные подпоследовательности и сливающий их в результирующий список. Работает с `List<T>`.

### 3\. Модель данных (Person)

Класс `Person` используется для демонстрации работы графа и сортировок.

  * Реализует интерфейс `IComparable<Person>`, выполняя сравнение объектов по полю `Age`.
  * Переопределяет `Equals` и `GetHashCode` для корректной работы в коллекциях.

## Тестирование

Проект `GraphAndSortingTests` использует фреймворк xUnit.

  * **Graph Tests:** Проверка создания, копирования, CRUD-операций с вершинами/ребрами, обработки исключений и корректности итераторов.
  * **Sorting Tests:** Проверка сортировки на пустых, одиночных, уже отсортированных и перевернутых коллекциях.
  * **Model Tests:** Проверка методов сравнения и эквивалентности класса `Person`.
